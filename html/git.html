Git

    git init — инициализировать новый репозиторий Git;
  • git clone — клонировать существующий репозиторий
  • git status — показать состояние репозитория (изменённые файлы, файлы в индексе и т. д.)
  • git add — добавить файл(ы) в индекс (подготовить к коммиту)
  • git commit -m "сообщение" — зафиксировать изменения (создать коммит с сообщением)
  • git log — посмотреть историю коммитов
  • git push — отправить локальные коммиты в удалённый репозиторий
  • git pull — получить изменения из удалённого репозитория и слить их с текущей веткой
  • git branch — показать список веток или создать новую ветку
  • git checkout — переключиться на другую ветку
  • git merge — слить указанную ветку с текущей
